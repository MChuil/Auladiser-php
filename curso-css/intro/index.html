Introducción a CSS

Cascade Style Sheets

Existe 3 maneras de agregar css

  1. Directamente a la etiqueta html usando el atributo style
  2. Dentro del head del mismo documento (parte superior)
  3. En un archivo aparte (.css)

Las reglas css, se caracterizan por ser un par propiedad: valor;

ejemplo:
color: #fff;

Las reglas css se aplican a las etiquetas html, a los ID y las Clases

Sintaxis: etiqueta{ propiedad:valor; propiedad:valor; propiedad:valor; }

id: identificador y no debe haber mas de uno igual.
El nombre del id no debe llevar espacios en blanco, ni caraceteres especiales. Solo string

Las clases son formas de agrupar elementos que puedan recibir las mismas reglas css, se utiliza el atributo: class
